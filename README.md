> 🚀 **Turn any spoken video into Urdu text** – Break language barriers and make video content accessible to Urdu speakers worldwide.

---

## 📖 Overview

**AudioTranscription** extracts audio from video files, transcribes the speech using AI, and translates it into **Urdu**. Perfect for:

- 📚 Educational lectures  
- 🎬 Movies & documentaries  
- 📰 News clips  
- 🎙️ Podcasts & interviews  
- 📢 Public speeches  

No matter the original language – English, Hindi, Arabic, Spanish, etc. – you get clean **Urdu text** output.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🎥 **Multi‑format video** | MP4, AVI, MOV, MKV, and more |
| 🧠 **Smart transcription** | Uses Whisper (OpenAI) for high accuracy |
| 🌐 **Auto language detection** | Recognizes source language automatically |
| ➡️ **Urdu translation** | Machine translation with contextual accuracy |
| 💾 **Flexible output** | Plain text (.txt) or subtitle (.srt) files |
| ⚡ **Fast processing** | Optimized for both CPU and GPU |
